## 设计参考关键词

- 任务游戏化：任务完成后获得经验、金币、物品等奖励。
- 属性/技能成长：把现实能力抽象成可升级属性，与任务完成绑定。
- 商店/仓库/开箱：将现实奖励、虚拟物品、随机奖励统一进激励经济系统。
- 成就系统：通过阶段目标、解锁条件和奖励反馈强化长期目标。
- 番茄钟：把专注行为转化为可奖励、可统计、可交易的游戏化资产。
- 高自定义：用户可调整属性、商品、奖励、清单、模块、API 联动。
- 离线优先与备份：数据主要在本地，提供 WebDAV/云盘/文件导出等备份路径。
