# 暗影君王智能体 (Shadow King Agent)

> 类似 Claude Code 的交互模式：理解上下文、调度 Skills、自适应决策

## 已实现

### ReAct 核心（App/ 目录）
- `ShadowAgent.swift` — 代理接口，自然语言聊天 + /command 命令路由 + 流式输出
- `AgentLoop.swift` — ReAct 循环（推理→工具调用→观察→再推理）
- `AgentTool.swift` — 10 个工具（查询 + 行动）
- `AgentMemory.swift` — 滑动窗口会话记忆（maxMessages=40），持久化到 `~/.config/solo-agent/memory.json`
- `SkillRegistry.swift` — 7 个内置技能（见下方 Skills 表）

### 对话 UI（Overlay/Views/ 目录）
- `ShadowAgentView.swift` — 主面板（聊天区 + 输入栏）
- `AgentChatView.swift` — 对话气泡列表
- `ChatInputBar.swift` — 输入框 + /command 补全

## Skills 系统（已实现）

| Skill | 说明 |
|-------|------|
| `/reorganize` | 合并今日碎片卡片 |
| `/analyze <batchId>` | 重新分析指定批次（Phase 1 + Phase 2） |
| `/regenerate <batchId>` | 用缓存转录重新生成卡片 |
| `/status` | 系统诊断：AI 配置、捕获状态、游戏引擎 |
| `/context` | 显示当前上下文信号 + 规则列表 |
| `/rule add/list/remove` | 管理上下文规则 |
| `/help` | 列出所有可用命令 |
| `/new` | 重置会话记忆（在 ShadowAgent 中处理） |

自然语言关键词（「整理」「状态」「上下文」「规则」）也可触发对应 Skill。

## 设计哲学

| Claude Code | 暗影君王智能体 |
|-------------|-----------|
| 读文件、搜索代码 | 读截图、分析视频 |
| 写代码、编辑文件 | 生成卡片、整理时间线 |
| 运行测试、构建 | 转录视频、调用 AI API |
| 理解项目上下文 | 理解用户当前活动上下文 |
| `/commit`, `/review` | `/reorganize`, `/analyze` |
