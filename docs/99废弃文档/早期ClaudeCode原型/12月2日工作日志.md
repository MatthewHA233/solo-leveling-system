## 📋 当前核心任务

**OCR优化方案 v1.1（ID初始化 + 常规监控）：**

**阶段一：初始化ID采集**
- 目标：采集满所有成员
- 滚动策略：无限上下滚动（直到采集完全）
- ID范围：160/162 等格式

**阶段二：双重略过机制**

1. **全局略过**（从CSV查询）
   - 读取 game_id_mapping.csv
   - Name 100% 匹配 → 直接略过此成员

2. **临近图片略过**（相邻帧检测）
   - 相邻图片OCR Name 100% 匹配 → 略过看ID动作
   - 节省耗时长的ID查看步骤

3. **异常数据收集**
   - 字符串不匹配但ID相同 → JSON记录
   - 为后续相似度算法提供基础

**核心约束：**
- 当前所有相似度判断都用 100% 字符串匹配
- 全局略过和临近略过共同作用（不冲突）
- 未来相似度算法会引用统一的匹配函数
