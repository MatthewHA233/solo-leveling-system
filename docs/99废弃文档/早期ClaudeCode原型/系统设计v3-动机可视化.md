# Solo Leveling System - 动机可视化

> 不是管理时间，是看清欲望

---

## 这不是什么

- ❌ 不是 GTD / 任务管理
- ❌ 不是习惯打卡
- ❌ 不是时间追踪
- ❌ 不是游戏化激励

这些东西市面上已经够多了。

---

## 这是什么

**一面镜子，让你看清：**

1. 你真正想要什么
2. 你现在离它多远
3. 你最近在靠近还是远离

---

## 核心交互

不是"今天有什么任务"。

而是：

```
┌─────────────────────────────────────────┐
│                                         │
│  你现在最想要的是什么？                  │
│                                         │
│  ① 让她见到我时心动                     │
│     ████████░░░░░░░ 55%                 │
│     ↑ 本周 +3%                          │
│                                         │
│  ② 不再为钱焦虑                         │
│     ███░░░░░░░░░░░░ 18%                 │
│     ↓ 本周 -2%                          │
│                                         │
│  ③ 能独立做出产品                       │
│     ██████░░░░░░░░░ 40%                 │

│                                         │

```

**就这些。**

没有任务列表。没有打卡。没有经验值。

只有：你想要什么，你离它多远，你在靠近还是远离。

---

## AI 的角色

AI 不管理你的时间。

AI 做两件事：

### 1. 帮你看清动机

```
用户: 最近很焦虑

AI: 你现在有三个在追的东西：
    - 让她心动（55%，在靠近）
    - 财务安全（18%，在远离）
    - 做产品（40%，停滞）

    焦虑可能来自第二个。上周发生了什么？
```

### 2. 帮你更新进度

```
用户: 项目交付了，拿到3000

AI: 财务安全：18% → 22%
    本月收入累计 8000，目标 20000
    还差 12000，还剩 28 天
```

## 动机的定义

动机不是目标，不是任务。

**动机是你内心的渴望，用最诚实的话说出来：**

| 动机 | 诚实的表达 |
|------|------------|
| 变帅 | "我想让她看到我时惊艳" |
| 赚钱 | "我不想再为钱低头" |
| 变强 | "我想成为能独立做事的人" |
| 修复关系 | "我不想带着遗憾" |

这些话可能不"正确"，但它们是真的。

---

## 进度怎么算

不是任务完成率。

**是距离目标的真实距离：**

```
动机：让她心动
├── 衡量标准：见面时的状态
├── 子维度：
│   ├── 体脂：25% → 15%（当前 55%）
│   ├── 穿搭：0 → 3套搭配（当前 33%）
│   └── 状态：焦虑 → 自信（当前 40%）
└── 综合进度：(55 + 33 + 40) / 3 = 43%
```

有些维度难以量化，就用主观感受 0-100。

---

## 进度怎么变

不是"完成任务 +5%"。

**是你告诉 AI 发生了什么，AI 帮你更新：**

```
用户: 今天跑了5公里

AI: 体脂从 24% 预估降到 23.9%
    【让她心动】55% → 55.3%
    继续保持，月底能到 58%
```

```
用户: 今天又刷了3小时手机

AI: 【做产品】本周无进展
    你上次推进是 4 天前
    要不要聊聊卡在哪？
```

## 与 AI 的对话

不是命令式的。是聊天：

```
你: 最近感觉没啥动力

AI: 看了下你的三个追求：
    - 让她心动：在稳步靠近
    - 财务：上周收入停了，在远离
    - 产品：4天没动了

    哪个最让你烦？

你: 钱吧，项目卡住了

AI: 游戏脚本那个？卡在哪？

你: 甲方不回消息

AI: 财务目标本月还差12000
    这个项目占多少？

你: 3000

AI: 那还有9000的缺口
    有别的收入来源吗？还是等这个？

你: 只有这个...

AI: 那要不要想个备选方案？
    万一这个黄了，怎么补这个缺口？
```

**AI 不告诉你"今天要做什么"，AI 帮你看清处境。**
