## 💪 变帅计划 v1.0

### 核心原理

要在 3 个月内出腹肌，需要：
1. **降体脂** - 腹肌是"瘦出来的"，体脂降到 15% 以下才能看见
2. **练核心** - 让腹肌有形状
3. **控饮食** - 七分吃三分练

### 每日训练计划（配合跳绳）

**时间安排：早起后 / 或晚饭前**（约 30-40 分钟）

#### 有氧：跳绳（已有习惯，继续保持）
- 目标：每天 2000 个 → 逐步提升到 3000 个
- 燃脂主力

#### 核心训练：瑜伽垫动作（每天 15-20 分钟）

| 动作 | 组数 x 次数 | 说明 |
|------|-------------|------|
| 卷腹 | 3 组 x 15 次 | 练上腹 |
| 抬腿 | 3 组 x 15 次 | 练下腹 |
| 俄罗斯转体 | 3 组 x 20 次 | 练侧腹（消除腰部赘肉） |
| 平板支撑 | 3 组 x 30-60秒 | 整体核心 |
| 登山跑 | 3 组 x 30秒 | 燃脂 + 核心 |

**组间休息：30-60 秒**

### 饮食原则（简单版）

- ❌ 戒掉/减少：奶茶、含糖饮料、零食、油炸、宵夜
- ✅ 多吃：蛋白质（鸡蛋、鸡胸肉、鱼）、蔬菜
- ⏰ 晚饭不要太晚，睡前 3 小时不吃东西

### 脸部发腮改善

- 降体脂后会自然改善
- 减少高钠食物（减少水肿）
- 多咀嚼（吃饭细嚼慢咽）
