# 当前项目 CLAUDE

这是仓库根目录 `CLAUDE.md` 的 Obsidian 桥接笔记，用来让产品考古索引可以走双向链接，而不是打开系统文件链接。

当前要点：

- 当前主线是 `clients/desktop`，也就是 Tauri 2 + React + Rust 桌面端。
- 旧 Python `server/` 和早期 SYSTEM 面板只能作为原型参考，不能当成当前实现状态。
- 没有明确要求时，不要自动运行 `npm run dev`、`cargo check`、`cargo build` 或发布构建。
- 做产品判断时，优先参考当前 `docs/第1章_产品愿景.md`、`docs/第3章_系统面板设计.md` 和 `docs/昼夜表/`。

如果需要修改开发规范，应该回到仓库根目录的 `CLAUDE.md` 原文件。
