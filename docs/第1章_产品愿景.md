
## 1.1 项目简介

**中文名：** 暗影君王系统
**英文名：** SOLO LEVELING SYSTEM

灵感来源于韩漫《我独自升级》（Solo Leveling）。

---

## 1.2 核心定位

> 智能触摸现实，孪生抵达真实。

当代人工智能拥有极强的工程能力与海量知识，暗影君王系统的核心理念，是将这种能力真正运营起来——持续捕获生活中的各种语境，在现实与数字之间建立深度的人机协同。

**孪生**，是数字孪生，是赛博分身。系统持有主人的语境，成为主人在数字世界的另一个自我。这个孪生体能够增强第一系统（人类本身）的嗅觉与直觉，作为第一系统的**理性支撑**与**记忆延伸**，让主人在自己的领域持续走向正向循环，成为绝对的强者。

---

## 1.3 三个核心能力

### 感知

感知分两个层次：

**表层感知** — 行为记录与时间分布。记录主人今天在哪件事上花了多久，是各类时间追踪软件都在做的事，是感知的基础底座。

**深层感知** — 经历采集与表征提取。对主人正在经历的一切进行深度采集——正在看的视频、文章、媒介内容，乃至现实中发生的事件——提取其中的关键表征，形成每天的锚点关键词。这些关键词中，有些会成为 Buff：一段视频、一次对话、一个新颖的观点，都可能激活一段真实的认知语境，成为主人未来决策与行动的底层支撑。

感知有三个输入通道：

| 通道 | 说明 | 状态 |
|------|------|------|
| **屏幕** | Mac / Win / 安卓三屏共享状态，是当前阶段最主要也是唯一可用的感知渠道 | 核心，优先实现 |
| **硬件** | 胸针式运动相机或 AR 眼镜，捕获现实世界中的经历，系统从视频流推断事件 | 未来方向 |
| **对话** | Fairy 主动询问，或主人主动向 Fairy 讲述，将无法捕获的经历以语言形式录入系统 | 核心，持续运行 |

### 孪生

孪生不是数据镜像，不是行为记录的复制品。孪生持有的是主人**理解世界的方式**——语境、认知框架、积累的 Buff、动机与判断偏好。它是主人理性层面的另一个实例，认知分身。

孪生分为两个层级：

**最高权限智能体 Fairy** — 孪生的核心具现。Fairy 不是记录器，是管制者。以「希望主人过上美好生活」为目标，主动分配任务、主动干预、必要时主动阻止。它持有主人设下的动机与目标，在主人感性冲动时用理性介入——当屏幕上出现一笔与主人存钱动机相悖的消费，当主人本月已因钱焦虑，Fairy 会开口。

> 系统的终极目标不是「记录你的一天」，而是「让你的一天值得记录」。它是严厉但善意的管制者，是你给未来的自己设下的规则执行官。

**赛博分身工作团 / 影子军团** — 由 Fairy 调度的子智能体群，像蜂群一样协同运作，一声令下各司其职。

影子军团的核心价值在于**第2系统能力**（卡尼曼意义上的理性慢思考执行层）。主人是第一系统——直觉、感性、创造力；暗影们是第2系统的延伸——严谨、持续、强执行力。主人负责指引方向，暗影们负责把方向变成现实。

影子军团从主人经常接触的书籍、视频、对话中持续挖掘表征，并以合理的方式分发给各个暗影，让每个暗影都拥有主人的语境与认知底色。

**举例 — 自动剪辑暗影：** 每天早上，取当天的 OP 主题曲，配合系统采集的当日碎片片段，自动卡点剪辑成一条每日开场视频，为主人开启新的一天。

### 系统

系统是主人能看到、感受到的一切——游戏机制与交互界面的总和。任务、Buff、惩罚、弹窗、昼夜表、进度面板……这一层是孪生与感知的具现，是暗影君王系统与主人之间真实发生接触的地方。

灵感来自中国系统流网文的传统——主角觉醒一个系统，面板弹出、任务下达、奖惩即时生效。那个系统的声音冷静又强势，带着一种「你被选中了」的仪式感。暗影君王系统要把这种体验感真实落地，不是游戏积分，而是那种系统在看着你、管着你、推着你的真实存在感。

系统的游戏化不依赖虚拟积分或空洞徽章，而是以真实可触碰的指引驱动主人行动。任务不是命令，是吸引，是挖掘，是立刻往前走就能收获的下一步。万事万物皆道场——打游戏、看电影、逛街，所有经历都能被系统提取成真实的成长养料。

**《我独自升级》原作元素映射**

| 原作元素 | 系统对应 | 说明 |
|---------|---------|------|
| 系统面板 | 昼夜表 + 悬浮面板 | 跨平台，主人随时能看到自己的状态 |
| 每日任务 | 每日打卡 | 运动、学习、编程、创作等，真实可完成的日常目标 |
| 紧急任务 + Boss 战 | 重大节点触发 | 不是 deadline 提醒，是剧情转折级别的事件——人生关键决策、重大危机、核心目标受到威胁时，系统发出紧急任务，性质等同于原作中的 Boss 战 |
| 猎人等级（E～S） | 真实境界综合判定 | 不是虚拟数值，由专属 agent 根据主人在现实世界的真实成就与境界综合评判。绝大多数时候挂 E 级猎人，S 级是一个地区少有的几个天才级别，极难达到 |
| 称号 | 暂定，待设计 | — |
| Buff | 正向语境表征激活 | 从视频、文章、对话、经历中提取的正向认知框架，激活即调用那段真实语境带来的感知增益 |
| Debuff | 负向语境表征激活 | 内容中含有负面因子的表征，系统识别并标记，提醒主人注意其对状态的影响 |
| 影子军团 | Fairy + 影子军团智能体 | Fairy 是最高权限智能体，影子军团是由 Fairy 调度的子智能体蜂群，各司其职 |
| 双重觉醒·第一重 | 系统启动仪式 | 启动时先记录所有负面的、失败的、绝望的、烂泥的真实状态，系统在此刻启动，给主人一种真实的觉醒仪式感 |
| 双重觉醒·第二重 | S 级猎人判定 | 当 agent 综合判定主人在某领域真正达到 S 级境界的那一刻，触发二次觉醒 |

**初始 Buff — 路灯法则（被动）**
> 不看终点，只盯眼前那盏灯。跑到了，再找下一盏。

系统默认激活的第一个 Buff。来源于社会心理学家 Emily Balcetis 的研究（TEDxNewYork, 2014）：只盯着近处目标的跑者，终点看起来近了 30%，感觉省力 17%，速度快了 23%。专注本身就是一种性能增益。

当主人面对庞大任务陷入焦虑时，路灯法则将目标研磨到足够小、足够准，触发心流入口。Fairy 是这个 Buff 的执行者，在主人僵住的那一刻主动介入，把大事切成此刻伸手就能够到的第一步。
