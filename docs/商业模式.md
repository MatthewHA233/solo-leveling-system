# 商业模式

## 定价

**328 元/月**（一天约 11 元，「一杯奶茶换一个 AI 管家」）

### 定价逻辑

- 不算便宜，能筛选出真正想改变的用户，留存率高
- 对标：Keep 会员 25/月、得到 365/年、知识星球 299-599/年——都是被动内容，而我们提供**全天候 AI 主动伴随**

---

## 成本结构（单用户/月）

| 项目 | 预估 |
|------|------|
| AI API（视频分析 + 文本生成 + 出题） | 200-500 元 |
| TTS（Fish TTS 语音播报） | 20-50 元 |
| 推送/后端基础设施 | 10-30 元 |

早期单用户可能微亏，规模化后 API 有议价空间。

---

## 核心卖点

**不是功能列表，是情感绑定：**

- 系统有固定声音、有性格、会生气会鼓励
- 用户续费不是因为功能好用，是因为「不想让系统失望」
- 本质上卖的是一段**关系**，不是一个工具

---

## 关键指标

**第一周留存**是生死线：

- 328 的价位没有容错空间，用户会非常挑剔
- 必须在第一周内让用户感受到不可替代的价值
